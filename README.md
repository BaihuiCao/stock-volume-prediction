# Stock Price & Volume Prediction

## Overview
This project analyzes historical S&P 500 stock price and trading volume data to identify patterns and predict future trading volume using machine learning models.

## Data
- Source: S&P 500 stock data from Kaggle
- Features include: Date, Open, High, Low, Close, Volume, Ticker

## Methods
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Machine learning models:
  - Random Forest Regressor
  - Baseline regression models
- Model evaluation using MAE and RMSE

## Key Results
- Random Forest significantly outperformed baseline models
- Volume-related lag features were strong predictors

## Tools
- Python
- pandas, numpy, scikit-learn
- matplotlib, seaborn
